Meu Portifólio Pessoal

Para este primeiro projeto, desenvolvi meu próprio Portfólio Web Pessoal e Dinâmico. Com base e foco em HTML5 e CSS3, busco demonstrar minhas habilidades, domínio e conhecimento em desenvolvimento frontend, design responsivo e experiência do usuário.

Funcionalidades Principais

  • **Navegação Intuitiva:** Estrutura de navegação clara e fácil de usar entre as seções Home, Currículo e Projetos.
  • **Experiência Visual Imersiva:** Uso de um vídeo de fundo dinâmico que se estende por todas as páginas para um impacto visual e engajamento do usuário.
  • **Design Responsivo Completo:** Otimização para visualização perfeita em diversos dispositivos, desde desktops e notebooks até tablets e smartphones.
  • **Página de Currículo Detalhada:** Apresentação organizada de informações pessoais, experiência, formação e habilidades, com opção de download do currículo em PDF.
  • **Galeria de Projetos Interativa:** Exibição dos projetos em um layout de cards flexíveis e responsivos, com miniaturas, títulos e botões "Ver Detalhes" para acesso rápido às informações.
  • **Conectividade Social:** Rodapé com links diretos para perfis profissionais (LinkedIn, GitHub) e pessoais (WhatsApp, Instagram).
  • **Páginas de Detalhes de Projetos:** Estrutura para cada projeto ter sua própria página aprofundada, com descrições, tecnologias e links relevantes (GitHub, Demonstração).

Objetivo do Projeto

Meu principal objetivo para este projeto, é usalo como base para apresentar futuros projetos e tambem usar como apresentação pessoal minha e de minhas habilidades em desenvolvimento.

Tecnologias Utilizadas

  • **HTML5:** Para a estruturação semântica do conteúdo.
  • **CSS3:** Para toda a estilização e design, com ênfase em Flexbox para layouts responsivos.
  • **Mídia:** Vídeos (MP4) para o plano de fundo e Imagens (PNG/JPG) para ícones e miniaturas.
  • **Controle de Versão:** Git e GitHub.

Mesmo sendo meu primeiro projeto, continuarei buscando melhorias para o mesmo, para que possa deixá-lo ainda mais interativo e responsivo com o usuário. Além disso, pretendo ir adicionando futuros outros projetos para deixá-lo mais completo.
